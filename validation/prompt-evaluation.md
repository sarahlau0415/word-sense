# Prompt Evaluation

## 当前状态

后端 prompt 已切到 Word Sense v2 的六维骨架。验证时不要只看“内容写得好不好”, 要看它是否稳定完成以下任务:

- 根据原句收窄 sense。
- 选择合适维度, 不硬凑六维。
- 把“感觉”和“跟中文比”写出产品差异。
- 不编造词源、人物、事件、论文或引语。
- 例句自然, 场域差异真实。

## 测试方法

每个测试词跑三种输入:

1. 只有词
2. 词 + 出处
3. 词 + 出处 + 原句

记录:

- 输出使用了哪些维度
- 是否承认不确定
- 是否有事实风险
- 是否有中文错位洞察
- 是否给出可迁移的“悟道时刻”

## 推荐测试集

```text
concerning
awkward
lowkey
vibe
agency
accountability
narrative
garment
shot
sweet
```

## 失败模式

- 把 sense 讲散, 像一篇百科。
- “跟中文比”只是在给翻译。
- “悟道时刻”变成空泛升华。
- 社媒例句靠堆流行词制造网感。
- 对词源过度确定。
- 用户给了原句, 但输出仍然像只输入了单词。
