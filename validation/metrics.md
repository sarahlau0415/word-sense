# Metrics

## 北极星指标

建议第一阶段使用:

> 每周产生“高质量词条体验”的目标用户数。

高质量词条体验定义:

- 用户输入了一个真实关心的词。
- 用户读完或停留足够久。
- 用户给出正向反馈、收藏、分享、订阅或再次查询。

不要一开始只看 PV。这个产品早期最重要的是“解释是否击中”, 不是泛流量。

## 使用漏斗

1. 访问首页
2. 输入词
3. 输入出处/原句
4. 生成词条
5. 阅读超过 60 秒
6. 保存/分享/订阅/再次查询

## 关键指标

| 指标 | 意义 | 早期目标 |
| --- | --- | --- |
| 查询转化率 | 首页是否让用户知道怎么用 | 40%+ |
| 语境填写率 | 用户是否愿意补充原句/出处 | 25%+ |
| 结果读完率 | 内容长度是否可接受 | 60%+ |
| 二次查询率 | 工具性是否成立 | 20%+ |
| 收藏/分享率 | 内容是否有传播价值 | 8%+ |
| 订阅转化率 | 是否能承接低频使用 | 5%+ |
| 7 日复访率 | 是否能形成习惯 | 10%+ |

## 质性信号

比数字更重要的早期信号:

- 用户说“这个词我以前真的理解错了”。
- 用户把解释转述给别人。
- 用户主动发来真实语境里的词。
- 用户问“能不能每天发我一个”。
- 用户愿意让它进入工作流: 看论文、写邮件、备课、做内容。

## 反向信号

需要警惕:

- 用户说“写得很好”, 但没有再次使用。
- 用户只把它当作知识读物, 不知道什么时候打开。
- 用户觉得输出太长, 但又说短了就没有价值。
- 用户无法说出它和 ChatGPT/词典的区别。
- 用户分享的是审美, 不是内容洞察。
